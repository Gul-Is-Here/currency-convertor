# Privacy Policy for CurrencyHub Live

**Last Updated:** December 11, 2025

## Introduction

CurrencyHub Live ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our mobile application (the "App"). Please read this privacy policy carefully.

## Information We Collect

### 1. Information You Provide to Us

**Personal Information**
- We do NOT collect any personally identifiable information (name, email, phone number, address) unless you explicitly provide it for support purposes.

**User Preferences**
- Currency preferences and favorites
- Notification settings
- Theme preferences (light/dark mode)
- Alert configurations for exchange rates
- Calculation history (stored locally on your device)

### 2. Automatically Collected Information

**Device Information**
- Device type and model
- Operating system version
- App version
- Unique device identifiers (for crash reporting only)

**Usage Data**
- App features used
- Time spent in the app
- Crash logs and error reports
- Network connectivity status

**Location Information**
- We do NOT collect precise location data
- We may detect your general region based on IP address for providing relevant currency suggestions

### 3. Third-Party Data Sources

**Exchange Rate Data**
- We fetch real-time exchange rates from **open.er-api.com** (Open Exchange Rates API)
- This API does not require authentication and does not collect personal information
- Data is used solely for displaying currency conversion rates

## How We Use Your Information

We use the collected information for the following purposes:

### Primary Functions
- Provide accurate currency conversion calculations
- Display real-time exchange rates and cryptocurrency prices
- Show historical rate charts and trends
- Send rate alert notifications (if enabled by you)
- Cache data for offline functionality

### App Improvement
- Analyze app usage to improve features and user experience
- Debug and fix crashes or errors
- Optimize app performance

### Notifications
- Send push notifications for rate alerts (only if you opt-in)
- Notify you about significant currency rate changes
- All notifications can be disabled in app settings

## Data Storage and Security

### Local Storage
- All your preferences, favorites, and calculation history are stored **locally on your device**
- We use encrypted SharedPreferences for sensitive data
- No personal data is transmitted to our servers

### Data Security
- We implement industry-standard security measures
- All API communications use HTTPS encryption
- We do not store payment information (the app is free)

### Data Retention
- Locally stored data remains on your device until you uninstall the app or clear app data
- Cached exchange rate data is automatically updated every 5 minutes
- You can clear all local data through app settings

## Permissions Required

Our app requests the following permissions:

### Internet Access (REQUIRED)
- **Purpose:** Fetch real-time exchange rates and cryptocurrency prices
- **Usage:** Data is downloaded but not uploaded

### Network State Access (REQUIRED)
- **Purpose:** Detect online/offline status to enable caching
- **Usage:** Ensures smooth offline functionality

### Notifications (OPTIONAL)
- **Purpose:** Send rate alert notifications
- **Usage:** Only if you enable rate alerts in settings
- **Control:** Can be disabled in device or app settings

### Wake Lock (OPTIONAL)
- **Purpose:** Keep screen on during rate monitoring
- **Usage:** Only when actively viewing live rates

### Boot Completed (OPTIONAL)
- **Purpose:** Restart rate monitoring after device reboot
- **Usage:** Only if you have active rate alerts

You have the following rights regarding your data:

### Access and Control

- Export your favorites and preferences
- Delete all local data by clearing app data or uninstalling

### Opt-Out Rights

- Turn off crash reporting (if available in settings)

### Data Deletion

- No data remains on our servers (we don't have servers for user data)
